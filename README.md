## 🏋️‍♀️ 서비스 소개
헬스 보조 파트너 매칭 어플리케이션 헬게더 입니다.

### 주요 기능
* 🧑‍🤝‍🧑 함께 운동할 파트너를 주변에서 구합니다.
* 💪 3대 운동(스쿼트, 벤치 프레스, 데드 리프트)을 인증해 파트너 모집에 지표로 삼습니다.
* 🎖️ 매일 운동인증을 통해 칭호를 부여합니다.

<br>
